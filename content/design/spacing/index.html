Spacing

{{ stache.config.product_name_short }} uses space between elements to separate content. Ample white space is critical to separate individual items and ensure that page content flows together. The base margin and base padding for {{ stache.config.product_name_short }} are both 10px, and multiples and fractions are available as necessary.

Styling values are for reference only, and you should not apply them directly. Instead, components should incorporate spacing as necessary.

For defined elements such as buttons, spacing is defined by the component. For new components, start with the base margin and base padding and adjust the spacing as necessary for readability and visual appeal.
