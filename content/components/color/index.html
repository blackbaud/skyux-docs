Color

As a general rule in SkyUX color is used to convey meaning, not as pure decoration. Colors should rarely need to be applied directly to an element; rather, components incorporate color as applicable. If a situation arises where color is needed outside of a component that provides it, there is likely a need for a new component to cover the scenario.

Percentage tints are used to lighten or darken these base colors as appropriate.
