Labels

While we use the Bootstrap convention for labels, our guideline is to use only four of the sub-classes as a way to convey the status of the label contents.

label-success

Success
<div class="label label-success">Success</div>
Usage

This is used to convey a good status.

label-info

Info
<div class="label label-info">Info</div>
Usage

This is used to convey a status that is neutral.

label-warning

Warning
<div class="label label-warning">Warning</div>
Usage

This is used to convey a status that indicates something may be going wrong and probably worth the user taking a deeper look.

label-danger

Danger
<div class="label label-danger">Danger</div>
Usage

This is used to convey a status that indicates something is very wrong and that the user should take action.
