Spacing

SKY UX uses space between elements to separate content. Ample white space is critical to separate individual items and ensure that page content flows together. The base margin and base padding for SKY UX are both 10px, and multiples and fractions are available as necessary.

Note that styling values are for reference only, and generally should not be applied directly. Instead, components should incorporate spacing as necessary.

For defined elements such as buttons spacing is described in the corresponding element documentation. For new components, start with the base margin and base padding, and adjust the spacing as necessary for readability and visual appeal.
