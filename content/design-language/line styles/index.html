Line styles

{{ stache.config.product_name_short }} uses various line styles to bound, divide, and highlight content. The lines are designed to maintain visual hierarchy regardless of how complex pages become or how many nested containers they include.

As with other types of formatting or decoration in {{ stache.config.product_name_short }}, use lines minimally. Remember that white space and positioning can group or highlight content without extra subdivisions or boundaries.

{{ stache.config.product_name_short }} uses a total of 6 line styles that can be applied via a SASS mixin:

  • A light separator to organize and divide blocks of content within a container
  • A dark separator to indicate borders and boundaries for components
  • 4 prescriptive, "isolated" line styles:
    • Navigation selection underline
    • Tab underline
    • Tile crown
    • Row separator

Generic styles

Light separator
Usage

The light separator is a very light gray, single-pixel line. It subdivides content within a container, but do not use it at the edge of a container. Only use it when it is adjacent to white space on both sides.

{{ stache.config.product_name_short }} currently uses this style for:

  • Subsection dividers within containers such as tiles
  • Top boundaries for the action bars on dialogues
Code
@include bb-border(light)
Styling
  • 1px solid #e7eaec

When this line style subdivides content within a container, use a 15px margin on both ends.

Dark separator
Usage

The dark separator is a moderate gray, single-pixel line. This heavier stroke holds up between two different colors, which makes it ideal for borders and boundaries. It also frequently separates active content from passive content. For example, it can separate a toolbar and the content that the toolbar acts on.

{{ stache.config.product_name_short }} currently uses this style for:

  • Borders on buttons
  • Top and bottom boundaries for toolbars
  • Column dividers for header rows in grids
  • Borders for cards
  • Built-in data visualization axes
Code
@include bb-border(dark)
Styling
  • 1px solid #ccc

When this line style bounds or divides an element horizontally or vertically, extend the line to the edges of the element.

Isolated styles

Isolated line styles are specific to certain parts of {{ stache.config.product_name_short }} and are defined as part of the component that they belong to. For the most part, they should not proliferate to other {{ stache.config.product_name_short }} elements.

Navigation selection underline
Usage

This line style highlights the active navigation area in the navigation menu. It aligns to the bottom of the navigation affordance and stretches to its full width.

Code
@include bb-border(nav)
Styling
  • 4px solid #1c84c6
Tab underline
Usage

This line style separates tab controls from their contents. It aligns to the bottom of the tab controls and stretches to the full width of the container.

Code
@include bb-border(tab)
Styling
  • 2px solid #1c84c6
Tile crown
Usage

This line affixes to the top of tiles. It extends above the top of the tile and stretches to its full width.

Code
@include bb-border(tile)
Styling
  • 4px solid #e7eaec
Row separator
Usage

This line separates members of a homogenous set. It is used between the rows in a grid, list, or repeater container. While the dotted style lowers its visual weight, the use of a moderate gray allows it to work with or without row striping on the list rows. It stretches the full width of the rows (not the width of the container that holds the rows).

Code
@include bb-border(row)
Styling
  • 1px dotted #ccc
